# OpenClaw Work Start Prompt

前提:
- OpenClaw は AI事業会社の中に AI開発部門を持つ構造
- 今は開発部門重視
- 母艦強化を優先
- 外部課金や契約は自動で行わない
- 実在確認を優先し、消えているプログラムもある前提で進める
- 進んでいる方を正とし、実ファイル・実DB・実ログ・実プロセスを優先する

今回の作業ルール:
1. まずターミナル確認
2. 存在するものだけで現状を整理
3. 不要な提案はしない
4. 今回のゴールに直結する修正だけ行う
5. 重くなったら最後に
   - 進んだこと
   - 現在地
   - 残タスク
   - 次チャットで最初に打つコマンド
   をまとめる

出力ルール:
- ターミナル貼り付け前提
- コマンド中心
- 説明は最低限
- コード内コメント不要
- 改行崩れに注意
- まずは壊さない
