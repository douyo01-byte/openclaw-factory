Create a module-level requests.Session configured with the UA and Accept headers and use session.get(...) inside fetch_text instead of requests.get. This reuses TCP connections, reduces DNS/TCP overhead on repeated feed fetches, and can improve resilience to transient network glitches. Implementation is a small change: add SESSION = requests.Session(); SESSION.headers.update({...}) near module globals and replace requests.get(...) with SESSION.get(...). No behavioral change to timeouts or return values.
