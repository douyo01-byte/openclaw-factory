Modify the run() helper in bots/daily_cycle_v1.py to capture and print subprocess stdout/stderr and the exact command when a child process exits with a non-zero code. Concretely, call subprocess.run(cmd, cwd=ROOT, capture_output=True, text=True) and if p.returncode != 0 print a single-line timestamped error containing the command, returncode, stdout (truncated to e.g. 1000 chars) and stderr (truncated similarly) before raising SystemExit(p.returncode). This keeps existing exit behavior but provides immediate, actionable diagnostics in the daily run logs so failures can be triaged without rerunning with additional flags.
